# C# Console Calculator

A simple console-based integer calculator that supports `+`, `-`, `*`, and `/`.  
The program validates input, handles negative numbers, and provides clean, formatted output for readability.

---

## Features
- Accepts input in the form: `operand operator operand` (e.g., `5 + 2`, `-3 * 10`)
- Handles extra spacing and negative numbers using regular expressions
- Continues running until the user types `exit`
- Clear, structured output formatting

---

## Skills Demonstrated
- Console I/O and basic user interaction
- Input validation using **regular expressions**
- Modular program design (methods for each operation)
- Control flow using loops and switch expressions

---

## How to Run
1. Clone or download this repository.
2. Open the project in Visual Studio or VS Code.
3. Build and run the project.
